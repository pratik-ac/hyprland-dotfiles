A clean and minimal Hyprland Wayland setup for Arch Linux, focused on productivity, aesthetics, and smooth workflow.<br><br>

This configuration includes a fully customized desktop environment built around Hyprland with tuned components for a modern tiling experience.<br><br>

✨ Features:<br>
🚀 Hyprland (Wayland compositor) as the core window manager<br>
🧩 Fully configured Waybar for status and system monitoring<br>
🎯 Rofi for app launcher and utilities<br>
💬 Dunst for lightweight notifications<br>
📁 Kitty terminal for fast and clean terminal experience<br>
🎨 GTK 3/4 + Qt5/Qt6 theming support (qt5ct, qt6ct, nwg-look)<br>
📋 Clipboard history support via cliphist<br>
🔧 Autostart scripts for smooth session initialization<br>
🎵 Spicetify integration for Spotify theming<br>
🖌️ xsettingsd for consistent X11 application theming<br><br>
