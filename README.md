A clean and minimal Hyprland Wayland setup for Arch Linux, focused on productivity, aesthetics, and smooth workflow.

This configuration includes a fully customized desktop environment built around Hyprland with carefully tuned components for a modern tiling experience.

✨ Features
🚀 Hyprland (Wayland compositor) as the core window manager
🧩 Fully configured Waybar for status and system monitoring
🎯 Rofi for app launcher and utilities
💬 Dunst for lightweight notifications
📁 Kitty terminal for fast and clean terminal experience
🎨 GTK 3/4 + Qt5/Qt6 theming support (qt5ct, qt6ct, nwg-look)
📋 Clipboard history support via cliphist
🔧 Autostart scripts for smooth session initialization
🎵 Spicetify integration for Spotify theming
🖌️ xsettingsd for consistent X11 application theming
